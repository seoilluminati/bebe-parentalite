# Coquillages d'allaitement : avis, utilisation et comparatif

Les coquillages d'allaitement (ou nursing shells) sont accessoires spécialisés pour allaitement : petits récipients rigides placés dans soutien-gorge, poitrine, entre allaitement. Accessoires mineur pour certaines, révolution pour d'autres. Ce guide explique véritablement ce qu'ils font (et ne font pas), et quand les utiliser vraiment.

## Qu'est-ce qu'un coquillage d'allaitement exactement ?

Coquillages d'allaitement sont petites coquilles creuses (formes diverses) conçues placer poitrine directement contact peau, sans contact vêtement.

**Matériaux courants** :
- **Silicone** : Flexible, confortable, plus courant
- **Plastique rigide** : Plus durable, moins flexible
- **Combinaison** : Silicone doux doublure + base plastique

## Fonctions réelles des coquillages

### Mythe vs réalité

**Mythe** : « Coquillages pomp le mamelon » → Faux. Aucune pompe action. Ils protègent seulement.

**Réalité** : Coquillages créent petit espace air entre poitrine vêtement = alleviation frottement peau.

### Fonction 1 : Protection des crevasses

Si vous avez crevasses ou douleurs extrême peau, coquillage crée barrière entre mamelon vêtement crevassé = réduction friction.

**Limite** : Coquillage n'accélère pas cicatrisation. C'est prévention friction, pas traitement.

**Consigne** : N'utiliser que si vraie douleur/crevasses. Pas besoin autrement.

### Fonction 2 : Collection léger lait

Pendant allaitement, l'autre sein coule parfois. Coquillages collectent lait perdu (sauf réellement perdu → prélèvement).

**Quantité** : Peut collecter 5-30ml par jour selon femme.

**Utilité** : Marginal pour quantité. Utile juste prévenir taches vêtement (confort, pas aliment).

### Fonction 3 : Mamelon inversé (inverted nipple)

Certaines femmes ont mamelon « inverted » (rentre dans poitrine). Coquillages portés plusieurs heures peuvent aider étirer légèrement mamelon.

**Efficacité** : Modérée. Aide certains cas, moins d'autres.

**Consigne** : Pédiatres/consultants reconnaissent aide possible, pas miracle.

## Quand vraiment utile ?

Coquillages bénéficient surtout :

1. **Crevasses mineures-modérés** (si douleur vraiment inconfortable)
2. **Mamelon inversé** (avant/pendant allaitement)
3. **Hypersensibilité extrême** (très rare)
4. **Prévention taches vêtement** (besoin pratique mineur)

La plupart femmes allaitement : zéro besoin coquillages. Excellente allaitement sans.

## Types de coquillages et modèles

### Coquillage silicone simple

Forme champignon simple, silicone doux.

**Avantages** :
- Très flexible, confortable
- Pas cher (10-25€)
- Léger, discret
- Facile nettoyage

**Inconvénients** :
- Pas très durable (silicone peut s'user)
- Peut tomber facilement dedans soutien-gorge
- Surface peut se coller (perte aération)

**Recommandé pour** : Besoin court-terme (crevasses temporaires).

### Coquillage rigide plastique

Forme dome ou plus structurée, plastique dur.

**Avantages** :
- Très durable
- Moins cher (8-15€)
- Bien aération (trous permettent circulation air)
- Reste en place soutien-gorge

**Inconvénients** :
- Moins confortable (rigide contre peau)
- Plus volumineux
- Peut créer pression inconfortable

**Recommandé pour** : Usage longue durée ou collection lait régulière.

### Coquillage avec airflow (aéré)

Plastique/silicone avec petits trous créant aération.

**Avantages** :
- Excelent aération (réduit humidité)
- Prévient moisi/infection
- Confortables généralement
- Bon équilibre dur/souple

**Inconvénients** :
- Coût plus élevé (20-40€)
- Moins de collection lait (lait peut couler trous)
- Moins efficace crevasses (air circulation = moins protection)

**Recommandé pour** : Prévention infection, aération optimale.

### Coquillage avec coussinets absorbants

Silicone + coussinets tampons absorbant lait.

**Avantages** :
- Collection efficace (coussinets absorbent)
- Prévention fuites vêtement
- Confortable doux

**Inconvénients** :
- Coussinets doivent changer régulièrement (déchets)
- Plus coûteux (25-50€)
- Moins ventilation (humidité possible)

**Recommandé pour** : Collection régulière importante lait, prévention taches.

## Installation correcte et utilisation

### Placement

1. Placez coquillage sur mamelon (ne doit pas pincer mamelon, juste reposer)
2. Attachez soutien-gorge. Coquillage doit rester place sans pression excessive
3. Vérifiez : Aucune douleur pincement. Coquillage pointe naturelle poitrine

### Portage

Certaines femmes portent coquillage tout jour entre allaitement. D'autres seulement certaines heures. Flexibilité.

**Recommandé** : Ne pas plus 8 heures jour (prévention infection/humidité)

### Entre allaitement

Collecte lait pendant jour (fuite entre allaitement). Peut être porté sans reproche.

**Alternative** : Coussinets allaitement absorb (réutilisables, moins invasif) accomplissent même fonction.

## Hygiène et nettoyage

**Quotidien** :
1. Videz lait collecté (dans pot stérile ou jeter)
2. Rincez eau tiède douce
3. Séchez complètement (tissu propre) avant rangement

**Hebdomadaire** :
1. Nettoyez eau chaude savon doux
2. Rincez complètement (pas résidu savon)
3. Stérilisez (eau bouillante 5 min, ou stérilisateur électrique)
4. Complètement sèches avant rangement

**Important** : Humidité = culture bactéries/moisissures. Séchage complet essentiel.

## Alternatives à considérer

### Coussinets d'allaitement réutilisables

Disques absorbants (tissu) placés soutien-gorge, absorbent fuites.

**vs coquillages** :
- Moins invasif
- Meilleur aération
- Lavables (zéro déchets)
- Moins cher long-terme
- Moins efficace protection crevasses

**Conclusion** : Meilleur choix prévention taches généralement. Pas pour vraies crevasses.

### Lanoline pure / pommade réparatrice

Pour crevasses vraies : traitement topique meilleur que coquillage protection.

**Avantage** : Traite le problème, pas seulement protège.

## Quand PAS utiliser coquillages

- **Absence vraie problème** : Zéro bénéfice
- **Allaitement difficile** : Consulter consultant lactation plutôt (coquillage masque problème sous-jacent)
- **Infection suspecte (mastite)** : Consulter médecin (coquillage peut empirer)

## Conclusion : utilitaire spécifique, pas universel

Les coquillages d'allaitement ne sont pas révolution marketing. C'est solution ciblée : si crevasses vraies ou mamelon inversé = peuvent aider. Pour majorité femmes allaitement ? Zéro besoin.

Conseil : Avant acheter : consultez consultant lactation. Expliquez situation. Eux recommanderont si utile vraiment. Coquillages sont accessoires allaitement, pas équipement central comme coussin ou tire-lait. Bon à avoir si besoin, pas fondamental autrement.
