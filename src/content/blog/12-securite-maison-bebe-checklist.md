# Sécuriser sa maison pour bébé : la checklist complète pièce par pièce

Les accidents à la maison sont la première cause de décès involontaires chez les enfants de 1-4 ans selon les autorités sanitaires. Pourtant, la plupart sont prévisibles et prévenables. Cette checklist détaillée, organisée pièce par pièce et par étape de développement, vous aide à identifier les risques et mettre en place les protections prioritaires.

## Sécurité générale : avant d'aller aux détails

### 0-6 mois : principalement sur parent

À ce stade, bébé ne se déplace pas seul. Priorités :
- Lit sécurisé (normes EN 716, pas d'objets)
- Surfaces plates/stables pour change/habillage
- Pas d'objets petits dans rayon (si venez de faire changement)
- Températures contrôlées (chauffage, AC)

**Temps sécurisation** : 1-2 heures (peu d'équipement nécessaire)

### 6-12 mois : roulement, puis motricité

Bébé roule, commence assis, puis rampe/se lève.
Priorités escalent rapidement :
- Barrières escaliers (haut ET bas si escaliers présents)
- Caches prises électriques
- Meuble antichute (ceux pouvant basculer fixés murs)
- Objets petits/menaçants enlevés rayons bas

**Temps sécurisation** : 4-8 heures (équipement modéré)

### 12-24 mois : marcheur autonome

L'enfant marche, court, monte escaliers avec soutien.
Tous risques précédents + :
- Sécurité cuisine aggravée (poêles, couteaux)
- Protection angles meubles
- Verrous armoires/tiroirs (si objets menaçants)
- Surveillance constante intensifiée

**Temps sécurisation** : 6-12 heures (équipement maximal)

## Chambre bébé

### Lit et sommeil

**Conforme normes** :
- ✓ Matelas ferme, bien ajusté (pas de vide entre matelas-lit)
- ✓ Barreaux 6.5 cm séparation maximum (prévient coincement tête)
- ✓ Pas de oreiller, douillet, tour de lit (risque étouffement)
- ✓ Gigoteuse ou draps seulement
- ✓ Barreau dégondre/mauvais état : réparer/remplacer

**Par âge** :
- 0-6 mois : Berceau ou cododo à côté lit parent (recommandé)
- 6-18 mois : Lit à barreaux standard
- 18+ mois : Passage possible lit petit enfant

### Température et ventilation

- ✓ Thermomètre chambre (température idéale 18-20°C)
- ✓ Humidité 40-60% (évite sécheresse/moisi)
- ✓ Fenêtre peut être ouverte (ventilation, pas climatisation extrême)
- ✓ Radiateur couvert grille (si accessible à enfant plus tard)

### Éclairage et surveillance

- ✓ Veilleuse très douce (ou apaisante uniquement après 6 mois)
- ✓ Babyphone placé haut, loin du lit
- ✓ Pas de cordons pendent (nœud autour cou-risque)
- ✓ Stores/stores sans cordons libres

### Objets et décoration

- ✓ Affichage murs : rien ne doit tomber
- ✓ Pas de objets petits sur rebords/meubles bas
- ✓ Plantes toxiques éloignées (ou dehors)
- ✓ Commodes/étagères fixées murs (si pas déjà)

## Salle de bain

### Eau et baignoire

- ✓ Thermostat mitigeur (température max 49°C idéalement)
- ✓ Tapis antidérapant baignoire/douche
- ✓ Baignoire bébé sécurisée ET stable
- ✓ Eau jamais laissée (baignoire ou seau) : risque noyade même 5 cm
- ✓ Porte baignoire peut fermer (gestion indépendante risque + tard)

### Équipements et produits

- ✓ Produits toxiques en hauteur ou verrouillés (savons/shampoings sauf si bébé)
- ✓ Médicaments armoire fermée
- ✓ Couches accessibles (parents, pas bébé)
- ✓ Pas d'objets électriques près eau (sèche-cheveux, radio)

### Électricité et ventilation

- ✓ Prises couvertes/protégées
- ✓ Interrupteur ventilation accessible (respiration hygiène)
- ✓ Pas de cordons électriques pendants

## Cuisine

### Chauffage et cuisson

- ✓ Grille protection devant poêle/four (si accessible)
- ✓ Casseroles/poêles manches tournées vers intérieur (pas accessible)
- ✓ Four verrou-protégé (6+ mois onwards)
- ✓ Lave-vaisselle verrou-protégé (acide, température)
- ✓ Café/eau chaude jamais sur rebord proche bébé

### Rangements dangereux

- ✓ Couteaux : tiroir verrouillé ou haut inaccessible
- ✓ Produits nettoyage : armoire fermée clé/verrou
- ✓ Verre cassé/brisé nettoyé immédiatement (sans bébé pièce)
- ✓ Sacs plastiques jetés ou rangés haut (risque asphyxie)

### Mobilier et accès

- ✓ Chaises hautes/tabourets : enfant 6+ mois ne peut grimper seul
- ✓ Tiroirs ne doivent pas se fermer sur doigts
- ✓ Frigorifique stable (pas basculement si enfant pend poignée)

## Salon/espace de vie

### Mobilier

- ✓ Meubles pouvant basculer fixés murs (TVs, étagères, commodes)
- ✓ Angles meubles bas : protections mousses (si enfant mobile)
- ✓ Pas de nappe qui pend (enfant peut tirer)
- ✓ Coussins canapé: pas sous bébé dormant (apnée-risque)

### Électricité et cordons

- ✓ Prises électriques couvertes/protégées
- ✓ Cordons chargeurs/TV enroulés haut (pas accessible)
- ✓ Pas de cordons pendent à niveaux tête enfant
- ✓ Lampadaires stables et loin circulation (pas basculement-risque)

### Objets hazardous

- ✓ Bouteilles alcohol éloignées/hautes
- ✓ Télécommandes non-toxiques si touchées (plastique sûr)
- ✓ Plantes toxiques éloignées complètement
- ✓ Pas de petits objets (pièces, perles, jouets adultes) sur tables basses

### Escaliers

- ✓ Barrière escaliers (haut ET bas si bébé mobile)
- ✓ Main courante : enfant peut saisir sans danger
- ✓ Pas d'objets sur escaliers (risque chute)
- ✓ Éclairage escalier bon (pas zones sombres)

## Chambres autres/bureau

### Rangements et électricité

- ✓ Cordons ordinateurs/bureaux rangés (pas accessible)
- ✓ Petits objets bureau (trombones, stylos) fermés
- ✓ Fournitures toxiques (colle, peinture) haut/fermé
- ✓ Prises protégées

### Meubles et rangements

- ✓ Bureaux/meubles hauts stables
- ✓ Classeurs verrous si poids-risque basculement

## Entrée/Porte d'entrée

- ✓ Porte d'entrée : enfant 2+ ans peut ouvrir (vérifiez)
- ✓ Verrous hauts (enfant ne peut pas sortir dehors seul)
- ✓ Pas de objets pointus près entrée
- ✓ Chaussures/manteaux rangés (pas sol où trébuchement)

## Extérieur/Jardin

### Escalade et basculement

- ✓ Piscine/bassin eau : clôture + alarme si possible
- ✓ Meubles jardin stables ou rangés quand non-utilisé
- ✓ Pas d'objets pointus accessibles
- ✓ Clôture jardin sécurisée (enfant ne peut pas sortir)

### Plantes et toxiques

- ✓ Plantes toxiques identifiées/éloignées (Ricin, Digitale, etc.)
- ✓ Engrais/pesticides fermés/haut
- ✓ Piscine/spa sécurisé (couverture, alarme)

## Équipements clés par étape

### 0-6 mois
- Lit sécurisé (EN 716)
- Babyphone
- Thermomètre chambre
- Température eau contrôlée

### 6-12 mois (AJOUTER)
- Barrière escaliers (haut/bas)
- Caches prises électriques
- Fixes murale (TV, meuble)
- Protections angles (si mobilité haute)
- Cordons rangés/inaccessibles

### 12-24 mois (AJOUTER)
- Verrous armoires (cuisine, produits)
- Protection four/poêle
- Grille protection radiateur
- Protections coins meubles (si non déjà)
- Barrière pièces hazard (salle de bain sans supervision)

## Checklist récapitative rapide

**0-6 mois** (30 min) :
- [ ] Lit conforme normes
- [ ] Chauffage 18-20°C
- [ ] Thermomètre chauffage eau
- [ ] Babyphone installé
- [ ] Plantes toxiques enlevées

**6-12 mois** (2 heures) :
- [ ] Barrières escaliers top/bottom
- [ ] Prises électriques protégées
- [ ] Meubles hauts fixés murs
- [ ] Cordons rangés hauteur
- [ ] Objets petits enlevés rayons bas

**12-24 mois** (2-3 heures) :
- [ ] Verrous armoires cuisine
- [ ] Protection four/poêle
- [ ] Angles meubles bas protégés
- [ ] Accès dangereux bloqués
- [ ] Surveillance constante intensifiée

## Conclusion : prévention = protection

La sécurisation maison peut sembler overwhelming, mais elle se fait progressivement, adaptée développement enfant. Les accidents graves sont généralement prévisibles : enfant monte escaliers sans barrière, ingère poison accidentellement, ou chute meuble non-fixé. Investir temps et petit argent en prévention (équipements, arrangements) sauve vies.

Utilisez cette checklist progressivement, adaptée votre situation spécifique. Pas besoin de tout d'un coup : priorisez risques plus imminents selon âge/mobilité enfant.
